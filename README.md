# Keypad-lock
### Description
A lock mechanism based on an ESP32 with a display on an OLED. It uses a keypad for user input and allows the user to set their own password. Once a password is set, the user has 3 attempts to get it right. If the user provides the right password they are taken to the menu, where they can either reset the password or quit. If a user fails to provide the correct password, they are denied access to the menu and have to wait for 10 seconds to attempt again.

This project is for HORIZONS EUROPA!

### Why I made this project
I wanted to try out the keypad since it was a more complex input device than push buttons, which I usually work with. I also see myself using passwords in my future projects so it was good practice. Also, I got more practice with an OLED display and now feel quite comfortable using it :)
